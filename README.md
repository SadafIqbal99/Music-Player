**🎵 MUSIC PLAYER – Vinyl Style Web Music Player
**
A modern vinyl-inspired web music player built using HTML, CSS, and JavaScript.
This project features a stylish dark/beige split layout with a rotating vinyl animation, playlist management, queue system, like feature, and category filtering.

**✨ Features**

🎶 Play / Pause / Next / Previous controls

💿 Rotating vinyl animation while music plays

❤️ Like system (saved using LocalStorage)

🔀 Shuffle & Repeat functionality

📂 Category filtering (Modern, Classic, 90s, Instrumental)

🔎 Live search by song title or artist

📜 Dynamic song list rendering

📌 Up Next queue system

🔊 Volume control with dynamic icon

📱 Responsive modern UI design

🖥️ UI Design

The design is inspired by a vinyl record player interface with:

Left Sidebar (Dark Theme)

Categories

Song list

Center Section

Rotating vinyl record

Currently playing song info

Play & Like buttons

Right Sidebar (Beige Theme)

“Up Next” queue

**🛠️ Technologies Used**

HTML5

CSS3 (Flexbox + Animations)

JavaScript (Vanilla JS)

Font Awesome Icons

LocalStorage API
